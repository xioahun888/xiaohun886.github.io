#  推送模板  

## 新增推送模版

进入订阅推送**【推送模块】** 列表，点击**【+模板】**
![image-20240711204838823](tuisongmoban.assets/image-20240711204838823.png) 

新增推送模板 
![img](tuisongmoban.assets\wps209.jpg)
填写信息，点击**【保存】**

## 隐形条件展示
推送模板ID和推送标题，会在订阅推送的**【通知日志查询】**列表中展示
